# Tilt Snake

Author: Jiwoo Yoo

Design: The classic game of snake...except you can move the apple. Tilting the board
moves your snake, but the apple is also affected by this.

Screen Shot:

![Screen Shot](screenshot.png)

How To Play:

WASD to tilt the board
